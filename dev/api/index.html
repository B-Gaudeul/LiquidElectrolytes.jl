Data of Electrolyte

LiquidElectrolytes.ElectrolyteData — Type
mutable struct ElectrolyteData <: AbstractElectrolyteData

Data for electrolyte.

  • nc::Int64

    Number of ionic species. Default: 2

  • iϕ::Int64

    Potential index in species list. Default: nc + 1

  • ip::Int64

    Pressure index in species list Default: nc + 2

  • T::Float64

    Temperature Default: (273.15 + 25) * ufac"K"

  • D::Vector{Float64}

    Mobility coefficient Default: fill(2.0e-9 * ufac"m^2/s", nc)

  • M0::Float64

    Molar weight of solvent Default: 18.0153 * ufac"g/mol"

  • M::Vector{Float64}

    Molar weight of ions Default: fill(M0, nc)

  • z::Vector{Int64}

    Charge numbers of ions Default: [(-1) ^ (i - 1) for i = 1:nc]

  • RT::Float64

    RT Default: ph"R" * T

  • F::Float64

    Faraday constant Default: ph"N_A*e"

  • c_bulk::Vector{Float64}

    Bulk ion concentrations Default: fill(0.1 * ufac"M", nc)

  • ϕ_bulk::Float64

    Bulk voltage Default: 0.0 * ufac"V"

  • p_bulk::Float64

    Bulk pressure Default: 0.0 * ufac"Pa"

  • Γ_bulk::Int64

    Bulk boundary number Default: 1

  • v0::Float64

    Molar volume of solvent Default: 1 / (55.4 * ufac"M")

  • κ::Vector{Float64}

    Solvation numbers Default: fill(10.0, nc)

  • v::Vector{Float64}

    Molar volumes of ions Default: fill(v0, nc)

  • ε::Float64

    Dielectric permittivity of solvent Default: 78.49

  • ε_0::Float64

    Dielectric permittivity of vacuum Default: ufac"ε0"

  • pscale::Float64

    Pressure scaling factor Default: 1.0e9

  • neutralflag::Bool

    Electroneutrality switch Default: false

  • scheme::Symbol

    Electroneutrality switch Default: :μex

  • epsreg::Float64

    Regularization parameter Default: 1.0e-20

LiquidElectrolytes.Cdl0 — Method
Cdl0(electrolyte)

Double layer capacitance at zero voltage for symmetric binary electrolyte.

Example

using LessUnitful
ely=ElectrolyteData(c_bulk=fill(0.01ufac"mol/dm^3"),2))
round(Cdl0(ely)|>u"μF/cm^2",digits=2)
# output

22.85 μF cm^-2
